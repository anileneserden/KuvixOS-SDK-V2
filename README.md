# KuvixOS SDK V2

Next-generation SDK for KuvixOS applications.

## Features

- C++ UI framework
- JSON-based UI layouts
- Widget system
- KEF application runtime
- Event-driven architecture

## Structure

- `include/` public headers
- `src/` runtime sources
- `examples/` sample apps
- `docs/` documentation
- `tools/` helper tools
